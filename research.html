0. VISIA肌診断の深掘り — なぜ診断が施術選定に不可欠なのか

「診断なしに施術を選ぶ」ことがいかに危険か。VISIAが数値化する8項目と、それぞれの数値が示す「あなたに必要な施術」の対応関係

VISIAが数値化する8つの肌指標

VISIAは世界109,300人（うち日本人11,000人）のデータベースと比較し、同世代平均を50%として各項目をスコアリングする。
「あなたの毛穴は同年代の上位30%に悪い」のような客観データが出る。これが「なんとなく」ではなく「数値に基づく処方」の根拠になる。

測定項目何がわかるかこの数値が悪いとき → 必要な施術
シミ 肌表面の色素沈着。肉眼で見えにくい薄いシミ・そばかすまで検出 光治療（IPL）、レーザートーニング、ビタミンC導入
隠れジミ（UV） まだ肌表面に出ていない紫外線ダメージ。将来のシミ予備軍 予防的ケア：トラネキサム酸内服、ピーリング、ケアシス（VC導入）
毛穴 毛穴の数・大きさ・形状。周辺皮膚との陰影差で判別 タイプ別に大きく異なる（下記の毛穴タイプ別参照）
シワ 表情じわ・静止じわの深さ・長さ・位置 ポテンツァ、ジュベルック（コラーゲン生成促進）
キメ（テクスチャ） 肌表面のなめらかさ・凹凸。キメの乱れや粗さ ピーリング、ハイドラフェイシャル、フラクショナルレーザー
赤み・色ムラ 毛細血管拡張、慢性的な炎症、酒さ傾向をクロス偏光撮影で可視化 IPL、ポテンツァ（赤みモード）、内服薬
ポルフィリン アクネ菌の代謝物を検出。皮脂分泌やニキビリスクを数値化 ハイドラフェイシャル（毛穴洗浄）、ケミカルピーリング、抗菌治療
メラニン / ヘモグロビン メラニンとヘモグロビンを分離画像化。色ムラの原因を特定 メラニン系 → 美白治療 / ヘモグロビン系 → 赤み治療（全く違うアプローチ）
診断なしの施術選択が危険な理由 — 肝斑の例 肝斑は見た目がシミに似ているが、原因が全く違う（表皮と真皮の境界部分の損傷）。
通常のシミ治療に使う高出力レーザーを肝斑に当てると、治療前よりも色が濃くなる（悪化する）。これは医学的に禁忌とされている。
肝斑の治療にはトラネキサム酸の内服 + ポテンツァ or フォトフェイシャルによる低侵襲アプローチが必要。

つまり「シミっぽいから○○」という自己判断やメニュー選択は、間違った施術で悪化させるリスクがある。
VISIAで「それがシミなのか肝斑なのか」を正確に判別してから施術を決めることが、安全かつ最短のルート。
出典: 山手皮膚科クリニック / タカミクリニック 肝斑治療

毛穴だけでも4タイプ — タイプを間違えると施術が効かない

「毛穴が気になる」と言っても、原因とタイプで必要な施術は全く異なる。VISIAの毛穴スコア + 医師の視診で正確にタイプを判別し、適切な施術を選定する。

詰まり毛穴（白い角栓）

原因：皮脂+古い角質が毛穴に詰まり、角栓が形成
VISIA所見：毛穴数多め、ポルフィリン値高め

  • → ハイドラフェイシャル（水流で角栓除去）
  • → ケミカルピーリング / ラヴィピール（角質溶解）
  • → ケアシス（ビタミンC導入で皮脂抑制）

※ ポテンツァ単体では詰まりは取れない。前処置としてハイドラが必須

黒ずみ毛穴（酸化・メラニン）

原因：角栓の酸化 or メラニン色素の沈着（2パターンある）
VISIA所見：毛穴スコア低+メラニン分離画像で判別

  • → 酸化型：ハイドラフェイシャル + 顔脱毛（産毛除去でトーンUP）
  • → メラニン型：ラヴィピール（メラニン対策）+ ケアシス（トラネキサム酸導入）

※ 汚れを取っても消えない黒ずみはメラニン沈着。原因の判別が重要

開き毛穴・すり鉢毛穴

原因：皮脂過剰分泌で毛穴が押し広げられた状態。すり鉢型は出口がクレーター状に凹む
VISIA所見：毛穴の陰影が深い、テクスチャスコア低

  • → ポテンツァ（RF引き締め、コラーゲン再構築）
  • → ジュベルック（PLLA+HAでコラーゲン生成促進）
  • → 重度：CO2フラクショナルレーザー（肌の再構築）

※ ポテンツァ+ジュベルックの鉄板コンビはこのタイプに最も効く

たるみ毛穴（加齢・弾力低下）

原因：コラーゲン・エラスチン減少で肌がたるみ、毛穴が楕円形に伸びる。毛穴同士がつながって見えることも
VISIA所見：シワスコア悪化+毛穴の形状が縦長

  • → ポテンツァ（真皮のコラーゲン再生）
  • → ジュベルック（長期的な弾力回復）
  • → HIFU / 糸リフト（たるみそのものへのアプローチ）

※ 毛穴洗浄だけでは絶対に改善しないタイプ。引き締め+弾力回復が必須

核心：「毛穴が気になる」の一言に、4つの全く違う正解がある 詰まり毛穴にポテンツァをやっても効果は薄い。たるみ毛穴にハイドラフェイシャルをやっても意味がない。
VISIAで毛穴のタイプ・深さ・数を数値化し、メラニン/ヘモグロビンの分離画像で原因を特定してから、初めて「あなたに最適な施術」が決まる。
これが「なんとなく人気の施術を受ける」のと「診断に基づくオーダーメイド」の決定的な違い。

出典: 大阪ほうれい線 毛穴タイプ解説 / 日比谷ヒフ科 すり鉢毛穴 / タカミクリニック 毛穴の種類 / La Clinique ポテンツァ×ハイドラ使い分け

診断→処方の具体例：同じ「毛穴が気になる」でもこう変わる

Case A：28歳女性

VISIA結果：毛穴スコア 35%（平均以下）/ ポルフィリン 72%（高い）/ メラニン 45%（正常）/ 隠れジミ 28%（少なめ）

医師の判断：詰まり+皮脂過剰型。アクネ菌活発。まず汚れを取り、皮脂コントロールしてからRF引き締め

処方：ハイドラフェイシャル → ラヴィピール → ケアシス（VC導入）

Case B：32歳女性

VISIA結果：毛穴スコア 28%（かなり悪い）/ シワ 40% / テクスチャ 30%（キメ乱れ）/ ポルフィリン 50%（正常）

医師の判断：開き+すり鉢型。汚れではなく構造の問題。コラーゲン再構築が必要

処方：ポテンツァ → ジュベルック → ケアシス（鎮静）

→ 同じ「毛穴が気になる」でも、VISIAの数値によって処方内容が完全に変わる。
→ LP上でこのような具体例を2〜3パターン見せることで、「自分はどっちなんだろう？診断受けてみたい」という欲求を引き出す。

その他の勝ち筋

1. 「パッケージ」ではなく「処方」と表現する
「パッケージ」という言葉はセット販売=売りつけのイメージがある（特商法の影響で警戒心が強い）。「あなたの肌に合わせた処方」「オーダーメイドプロトコル」等の表現に変えるだけで、印象が「商品」から「医療」に変わる。
2. 人気施術の効果を「爆上げする」相乗効果を訴求する
「ポテンツァ単体」ではなく「なぜこの順番で受けると効果が2〜3倍変わるのか」を論理的に説明する。ハイドラで毛穴洗浄→ポテンツァでRF→ジュベルック導入→ケアシス鎮静。各ステップに理由がある。
3. 次世代機器の選択肢で差別化する
ポテンツァだけでなくエリシスセンス・ブレッシングも提案できること。PKG2ではサブシジョン+キュアジェットの組み合わせ。「診断結果に基づいて最適な機器まで選定する」と伝えることで、他院との決定的な差になる。
4. VISIAのビフォーアフターを「数値」で見せる
写真のビフォーアフターは「加工疑惑」がつきまとう。VISIAの数値（毛穴数、シミ面積等）は客観的データとして信頼度が高い。「毛穴数 342 → 187」のような具体数値が最も説得力がある。
5. DT・痛みの正直な開示 = 信頼
パーソナル条件（DT許容度・痛み耐性）を考慮してプランを組むことを明示する。「無理のないスケジュール」「あなたの生活に合わせた治療計画」。特にPKG2はDTが長い施術が多いため、正直な開示が経験者の信頼を獲得する最大の武器。
